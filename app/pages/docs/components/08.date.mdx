# Date

<MetaDescription>
The Date component provides a comprehensive date and time picker with a calendar popover interface. It supports date selection, optional time selection, disabled date ranges, highlighted dates, and flexible formatting options.
</MetaDescription>

## Basic date picker

The Date component supports v-model for two-way data binding with an `IkTime` instance.

```vue title="[[_Source code_]]" preview
<template>
  <IkDate v-model="selectedDate" placeholder="Choose date..." class="ik-mb-4" />
  <p v-if="selectedDate">Selected: {{ selectedDate.format('ddd, D MMM YYYY') }}</p>
</template>
<script setup>
import { IkDate } from '@ikol/ui-kit/components/IkDate';
import { time } from '@ikol/ui-kit/classes/dates';
import { ref } from 'vue';

const selectedDate = ref(null);
</script>
```

### Disable date ranges

Use the `disabled_datespans` prop to disable specific date ranges. This accepts an array of `IkTimeSpan` objects.

```vue title="[[_Source code_]]" preview
<template>
  <IkDate v-model="rangeDate" :disabled_datespans="disabledRanges" placeholder="Some dates disabled..." class="ik-mb-4" />
  <p v-if="rangeDate">Selected: {{ rangeDate.format('D MMM YYYY') }}</p>
</template>
<script setup>
import { IkDate } from '@ikol/ui-kit/components/IkDate';
import { IkTimeSpan, time } from '@ikol/ui-kit/classes/dates';
import { ref } from 'vue';

const rangeDate = ref(null);

// Disable dates from 3 days ago to 3 days from now
const disabledRanges = [
  new IkTimeSpan({
    from: time().subtract(3, 'days'),
    to: time().add(3, 'days')
  })
];
</script>
```

## Events

### update:modelValue

Emitted when the selected date changes. The event payload is an `IkTime` instance or `null`.

```vue title="[[_Source code_]]" preview
<template>
  <IkDate v-model="eventDate" @update:modelValue="onDateChange" placeholder="Choose date..." class="ik-mb-4" />
  <p v-if="lastChange">Last change: {{ lastChange }}</p>
</template>
<script setup>
import { IkDate } from '@ikol/ui-kit/components/IkDate';
import { ref } from 'vue';

const eventDate = ref(null);
const lastChange = ref('');

const onDateChange = (date) => {
  lastChange.value = date ? `Date changed to: ${date.format('D MMM YYYY')}` : 'Date cleared';
};
</script>
```

### update:month

Emitted when the user navigates to a different month in the calendar. The event payload is an `IkTime` instance representing the first day of the displayed month.

```vue title="[[_Source code_]]" preview
<template>
  <IkDate v-model="monthDate" @update:month="onMonthChange" placeholder="Choose date..." class="ik-mb-4" />
  <p v-if="currentMonth">Current month: {{ currentMonth }}</p>
</template>
<script setup>
import { IkDate } from '@ikol/ui-kit/components/IkDate';
import { ref } from 'vue';

const monthDate = ref(null);
const currentMonth = ref('');

const onMonthChange = (month) => {
  currentMonth.value = month.format('MMMM YYYY');
};
</script>
```

## Methods

The Date component exposes methods that can be accessed via template refs.

### focus()

Focuses the day input and opens the calendar popover.

```vue title="[[_Source code_]]" preview
<template>
  <IkDate ref="dateRef" v-model="methodDate" placeholder="Choose date..." class="ik-mb-4" />
  <IkButton @click="focusDate">Focus Date Picker</IkButton>
</template>
<script setup>
import { IkDate } from '@ikol/ui-kit/components/IkDate';
import { IkButton } from '@ikol/ui-kit/components/IkButton';
import { ref } from 'vue';

const dateRef = ref(null);
const methodDate = ref(null);

const focusDate = () => {
  dateRef.value?.focus();
};
</script>
```

### focusDay()

Focuses the day input specifically.

### focusHour()

Focuses the hour input and opens the hour picker popover (when time selection is enabled).
