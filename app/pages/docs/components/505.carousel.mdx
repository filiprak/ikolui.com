# Carousel

<MetaDescription>
The Carousel component provides a powerful, flexible carousel/slider built on Swiper.js. It supports navigation arrows, pagination indicators, autoplay, infinite looping, grid layouts, zoom functionality, and pagination with infinite scroll. Perfect for displaying image galleries, product showcases, testimonials, or any collection of content.
</MetaDescription>

## Basic carousel

The Carousel component displays slides in a horizontal scrolling container. Use `IkCarouselSlide` components as children.

```vue title="[[_Source code_]]" preview
<template>
  <IkCarousel>
    <IkCarouselSlide>
      <div style="background: var(--bg-primary-regular-default); padding: 48px; text-align: center; color: white;">
        Slide 1
      </div>
    </IkCarouselSlide>
    <IkCarouselSlide>
      <div style="background: var(--bg-success-regular-default); padding: 48px; text-align: center; color: white;">
        Slide 2
      </div>
    </IkCarouselSlide>
    <IkCarouselSlide>
      <div style="background: var(--bg-error-regular-default); padding: 48px; text-align: center; color: white;">
        Slide 3
      </div>
    </IkCarouselSlide>
  </IkCarousel>
</template>
<script setup>
import { IkCarousel, IkCarouselSlide } from '@ikol/ui-kit/components/IkCarousel';
</script>
```

## Pagination

Enable pagination with infinite scroll using the `paginate` prop. The carousel will emit `ik-load-more` events when the loader slide becomes visible.

```vue title="[[_Source code_]]" preview
<template>
  <div style="display: flex; flex-direction: column; gap: 16px;">
    <IkCarousel paginate @ik-load-more="loadMore">
      <IkCarouselSlide v-for="slide in slides" :key="slide">
        <div style="background: var(--bg-primary-regular-default); padding: 48px; text-align: center; color: white;">
          {{ slide }}
        </div>
      </IkCarouselSlide>
    </IkCarousel>
    <div>Loaded {{ slides.length }} slides</div>
  </div>
</template>
<script setup>
import { IkCarousel, IkCarouselSlide } from '@ikol/ui-kit/components/IkCarousel';
import { ref } from 'vue';

const slides = ref(['Slide 1', 'Slide 2', 'Slide 3']);

function loadMore() {
  const newSlides = Array.from({ length: 3 }, (_, i) => `Slide ${slides.value.length + i + 1}`);
  slides.value.push(...newSlides);
}
</script>
```

## Slide change event

Listen to the `ik-slide-change` event to track the current slide index.

```vue title="[[_Source code_]]" preview
<template>
  <div style="display: flex; flex-direction: column; gap: 16px;">
    <IkCarousel @ik-slide-change="handleSlideChange">
      <IkCarouselSlide><div style="background: var(--bg-primary-regular-default); padding: 48px; text-align: center; color: white;">Slide 1</div></IkCarouselSlide>
      <IkCarouselSlide><div style="background: var(--bg-success-regular-default); padding: 48px; text-align: center; color: white;">Slide 2</div></IkCarouselSlide>
      <IkCarouselSlide><div style="background: var(--bg-error-regular-default); padding: 48px; text-align: center; color: white;">Slide 3</div></IkCarouselSlide>
    </IkCarousel>
    <div>Current slide: {{ currentSlide + 1 }}</div>
  </div>
</template>
<script setup>
import { IkCarousel, IkCarouselSlide } from '@ikol/ui-kit/components/IkCarousel';
import { ref } from 'vue';

const currentSlide = ref(0);

function handleSlideChange(index) {
  currentSlide.value = index;
}
</script>
```
