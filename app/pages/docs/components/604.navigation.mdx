# Navigation

<MetaDescription>
The Navigation component provides a responsive sidebar navigation that supports both full and mini (collapsed) modes. It automatically adapts to mobile devices with slide-in behavior and backdrop overlay, and integrates seamlessly with the App component for coordinated navigation management. Perfect for creating main application navigation, sidebars, and hierarchical navigation structures.
</MetaDescription>

## Basic navigation

The Navigation component creates a sidebar navigation panel. On desktop, it can be toggled between full and mini modes. On mobile, it slides in from the left when opened.

```vue title="[[_Source code_]]" preview client_only client_only
<template>
  <div style="position: relative; height: 400px; border: 1px dashed var(--border-neutral-regular-default);">
    <IkNavigation v-model:open="navOpen" v-model:mini="navMini" style="position: absolute">
      <IkListItem link="#" class="ik-mb-2">
        <template #prepend>
          <IkIcon name="home" />
        </template>
        Dashboard
      </IkListItem>
      <IkListItem link="#" class="ik-mb-2">
        <template #prepend>
          <IkIcon name="folder" />
        </template>
        Projects
      </IkListItem>
      <IkListItem link="#" class="ik-mb-2">
        <template #prepend>
          <IkIcon name="users" />
        </template>
        Team
      </IkListItem>
      <IkListItem link="#" class="ik-mb-2">
        <template #prepend>
          <IkIcon name="cog" />
        </template>
        Settings
      </IkListItem>
    </IkNavigation>
    <div style="padding: var(--s-6);">
      <IkButton @click="navOpen = !navOpen" class="ik-mb-4">Toggle Navigation</IkButton>
      <IkButton @click="navMini = !navMini">Toggle Mini Mode</IkButton>
    </div>
  </div>
</template>
<script setup>
import { IkNavigation } from '@ikol/ui-kit/components/IkNavigation';
import { IkListItem } from '@ikol/ui-kit/components/IkList';
import { IkIcon } from '@ikol/ui-kit/components/IkIcon';
import { IkButton } from '@ikol/ui-kit/components/IkButton';
import { ref } from 'vue';

const navOpen = ref(true);
const navMini = ref(false);
</script>
```

## Navigation with top and bottom slots

Use the `top` and `bottom` slots to add header content (like logos or branding) and footer content (like user profile or actions) to the navigation.

```vue title="[[_Source code_]]" preview client_only
<template>
  <div style="position: relative; height: 500px; border: 1px dashed var(--border-neutral-regular-default);">
    <IkNavigation v-model:open="navOpen" v-model:mini="navMini" style="position: absolute">
      <template #top>
        <div class="ik-pa-6 ik-text-center">
          <div class="ik-mb-4" style="font-size: 24px; font-weight: bold;">Logo</div>
          <div style="font-size: 12px; opacity: 0.7;">Application Name</div>
        </div>
      </template>
      
      <IkListItem link="#" class="ik-mb-2">
        <template #prepend>
          <IkIcon name="home" />
        </template>
        Dashboard
      </IkListItem>
      <IkListItem link="#" class="ik-mb-2">
        <template #prepend>

        </template>
        Projects
      </IkListItem>
      <IkListItem link="#" class="ik-mb-2">

      <template #bottom>
        <div class="ik-pa-6 ik-text-center" style="border-top: 1px solid var(--border-neutral-regular-default);">
```

## Secondary navigation

Use the `secondary` prop to create nested or secondary navigation panels that don't register with the App component and have different styling behavior.

```vue title="[[_Source code_]]" preview client_only
<template>
  <div style="position: relative; height: 400px; border: 1px dashed var(--border-neutral-regular-default); display: flex;">
    <IkNavigation v-model:open="navOpen" v-model:mini="navMini" style="position: relative; height: 100%;">
      <IkListItem link="#" class="ik-mb-2">
        <template #prepend>
          <IkIcon name="home" />
        </template>
        Main Item 1
      </IkListItem>
      <IkListItem link="#" class="ik-mb-2">
        <template #prepend>
          <IkIcon name="folder" />
        </template>
        Main Item 2
      </IkListItem>
    </IkNavigation>
    
    <IkNavigation v-model:open="secondaryOpen" secondary style="position: relative; height: 100%; border-left: 1px solid var(--border-neutral-regular-default);">
      <IkListItem link="#" class="ik-mb-2">
        Sub Item 1
      </IkListItem>
      <IkListItem link="#" class="ik-mb-2">
        Sub Item 2
      </IkListItem>
      <IkListItem link="#" class="ik-mb-2">
        Sub Item 3
      </IkListItem>
    </IkNavigation>
    
    <div style="padding: var(--s-6); flex: 1;">
      <IkButton @click="navMini = !navMini" class="ik-mb-4">Toggle Main Mini</IkButton>
      <IkButton @click="secondaryOpen = !secondaryOpen">Toggle Secondary</IkButton>
    </div>
  </div>
</template>
<script setup>
import { IkNavigation } from '@ikol/ui-kit/components/IkNavigation';
import { IkListItem } from '@ikol/ui-kit/components/IkList';
import { IkIcon } from '@ikol/ui-kit/components/IkIcon';
import { IkButton } from '@ikol/ui-kit/components/IkButton';
import { ref } from 'vue';

const navOpen = ref(true);
const navMini = ref(false);
const secondaryOpen = ref(true);
</script>
```

## Long content with scrolling

The Navigation component automatically wraps content in a scrollable area, making it perfect for long navigation lists.

```vue title="[[_Source code_]]" preview client_only
<template>
  <div style="position: relative; height: 400px; border: 1px dashed var(--border-neutral-regular-default);">
    <IkNavigation v-model:open="navOpen" v-model:mini="navMini" style="position: absolute">
      <IkListItem link="#" class="ik-mb-2" v-for="i in 20" :key="i">
        <template #prepend>
          <IkIcon name="circle" />
        </template>
        Navigation Item {{ i }}
      </IkListItem>
    </IkNavigation>
    <div style="padding: var(--s-6);">
      <IkButton @click="navMini = !navMini">Toggle Mini Mode</IkButton>
    </div>
  </div>
</template>
<script setup>
import { IkNavigation } from '@ikol/ui-kit/components/IkNavigation';
import { IkListItem } from '@ikol/ui-kit/components/IkList';
import { IkIcon } from '@ikol/ui-kit/components/IkIcon';
import { IkButton } from '@ikol/ui-kit/components/IkButton';
import { ref } from 'vue';

const navOpen = ref(true);
const navMini = ref(false);
</script>
```
