# Grid

<MetaDescription>
The Grid component provides a powerful 12-column grid layout system using CSS Grid. It supports responsive breakpoints, container queries for relative sizing, and flexible spacing options. Perfect for creating complex, responsive layouts.
</MetaDescription>

## Basic grid

The Grid component creates a 12-column grid layout. Grid items span all 12 columns by default.

```vue title="[[_Source code_]]" preview
<template>
  <IkGrid class="ik-pa-4" style="border: 1px dashed var(--border-neutral-regular-default);">
    <div class="ik-grid-item ik-pa-2" style="background: var(--bg-primary-regular-default);">Item 1</div>
    <div class="ik-grid-item ik-pa-2" style="background: var(--bg-primary-regular-default);">Item 2</div>
    <div class="ik-grid-item ik-pa-2" style="background: var(--bg-primary-regular-default);">Item 3</div>
  </IkGrid>
</template>
<script setup>
import { IkGrid } from '@ikol/ui-kit/components/IkGrid';
</script>
```

## Relative grid (container queries)

Use the `relative` prop to enable container-based responsive breakpoints. This uses CSS container queries instead of viewport media queries, making the grid responsive to its container size rather than the viewport.

When using `relative`, you must provide breakpoint values via props (`xs`, `sm`, `md`, `lg`) which define the minimum container widths for each breakpoint.

```vue title="[[_Source code_]]" preview
<template>
  <div style="resize: both; overflow: auto; border: 2px solid var(--border-neutral-regular-default); padding: 16px; min-width: 200px; min-height: 200px;">
    <IkGrid relative :xs="300" :sm="500" :md="700" :lg="900" spacing_x="3" class="ik-pa-4" style="border: 1px dashed var(--border-neutral-regular-default);">
      <div class="ik-grid-item ik-grid-item--col-xxs-12 ik-grid-item--col-xs-6 ik-grid-item--col-sm-4 ik-grid-item--col-md-3 ik-pa-2" style="background: var(--bg-primary-regular-default);">
        Item 1
      </div>
      <div class="ik-grid-item ik-grid-item--col-xxs-12 ik-grid-item--col-xs-6 ik-grid-item--col-sm-4 ik-grid-item--col-md-3 ik-pa-2" style="background: var(--bg-primary-regular-default);">
        Item 2
      </div>
      <div class="ik-grid-item ik-grid-item--col-xxs-12 ik-grid-item--col-xs-6 ik-grid-item--col-sm-4 ik-grid-item--col-md-3 ik-pa-2" style="background: var(--bg-primary-regular-default);">
```

## Click handler

Use the `onClick` prop to handle click events on the grid container.

```vue title="[[_Source code_]]" preview
<template>
  <IkGrid 
    class="ik-pa-4" 
    style="border: 1px dashed var(--border-neutral-regular-default); cursor: pointer;"
    :onClick="handleClick"
  >
    <div class="ik-grid-item ik-grid-item--col-xxs-6 ik-pa-2" style="background: var(--bg-primary-regular-default);">Click me</div>
    <div class="ik-grid-item ik-grid-item--col-xxs-6 ik-pa-2" style="background: var(--bg-primary-regular-default);">Item 2</div>
  </IkGrid>
  <p v-if="message" class="ik-text--sm ik-text--default-light ik-mt-4">{{ message }}</p>
</template>
<script setup>
import { IkGrid } from '@ikol/ui-kit/components/IkGrid';
import { ref } from 'vue';

const message = ref('');

const handleClick = () => {
  message.value = 'Grid container clicked!';
  setTimeout(() => message.value = '', 2000);
};
</script>
```
